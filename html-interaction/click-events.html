Click Events

In JavaScript, code may be executed on demand by using Functions and Event Listeners.

Functions

A function is a named block of code.

The code stored inside of a function will not execute until the function is invoked / called (ie. told to do its job!)

To create a function we can use a function declaration:


// Declare a function that stores code
// The name of this function is greeting
function greeting() {
  alert("Hello Functions");
  alert("I am inside of a function.");
}

The function keyword goes first, then the name of the function followed by parentheses. Finally the code of the function, "the function body", goes between curly braces.

The name of a function should represent the task of the function.

Function names should follow the same rules and conventions as variable names.

Event Listeners

Event listeners may be added to HTML elements.

Event listeners will tell a specified function to do its job when a certain event occurs.

Event Listener Syntax

element.addEventListener("event", nameOfFunction);

index.html


<body>
<h1 id="headingId">This is a Heading</h1>
<p><button id="btnId">Click Me</button></p>

<!-- Run External JavaScript File -->
<script src="script.js"></script>
</body>

script.js

The code below will run the greeting function, declared above, when any part of the web page is clicked.


// Run greeting function when document is clicked
document.addEventListener("click", greeting);

The code below will run the greeting function when the button element is clicked.


// Variable to store reference to button element
let btnEl = document.getElementById("btnId");

// Run greeting function when button is clicked 
btnEl.addEventListener("click", greeting);

Summary

  • Functions may be used to store blocks of code.
  • Event listeners may be added to html elements. When the specified event occurs, the event listener will tell the provided function to run its code.
