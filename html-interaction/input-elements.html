<input> Elements

HTML <input> Element

An <input> element will display as an input-box on a web page: .

An <input> element may have id and value attributes.

  • id attribute: use getElementById() to get a reference to the element.
  • value attribute: may be used to initialize the value of the input element.

index.html


<p>Enter your name: <input id="nameId"></p>
<p>Enter the time of day: <input id="timeId" value="Morning"></p>
<p><button id="btnId">Submit</button></p>

Output of index.html

Enter your name:

Enter the time of day:

Get Value of <input> Element

The following steps may be used to get the value of an <input> element.

  1. Use the id of the <input> element to get a reference to the element. 
    2. let nameEl = document.getElementById("nameId");
  2. Use the value property to get the value of the <input> element and store the result in a variable for future use. 
    3. let name = nameEl.value
  3. Use the variable that is storing the user input.
    4. alert(`Hello ${name}.`);

script.js


// HTML variables
let btnEl = document.getElementById("btnId");
let nameEl = document.getElementById("nameId");
let timeOfDayEl = document.getElementById("timeId");

// Add Event Listener to button element
btnEl.addEventListener("click", encourage);

function encourage() {
  // Input 
  let name = nameEl.value;
  let timeOfDay = timeOfDayEl.value;

  // Output
  alert(`Hello ${name}.`);
  alert(`Have a great ${timeOfDay}!`);
}

Attention

  • be sure to use an event listener to give the user time to type something into the input element!
  • the value of the input element is always a String data type. Convert to a Number data type if required.

Summary

  • JavaScript can get the value of <input> elements.
  • The value of an <input> element is always a String.
