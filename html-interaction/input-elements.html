<input> Elements

Topic Explanation

An <input> element will display as an input-box on a web page: .

An <input> element is a self-closing element: it only has an opening tag and no closing tag.

An <input> element may have id and value attributes.

  • id attribute: use getElementById() to get a reference to the element.
  • value attribute: use to initialize the value of the input element.

index.html


<p>Enter your name: <input id="name-in"></p>
<p>Enter the time of day: <input id="time-in" value="Morning"></p>
<p><button id="btn">Submit</button></p>

Use JavaScript to select an <input> element by id and then use its value property to get the contents of the input-box.

script.js


// HTML variables
let btnEl = document.getElementById("btn");
let nameInEl = document.getElementById("name-in");
let timeOfDayEl = document.getElementById("time-in");

// Add Event Listener to button element
btnEl.addEventListener("click", encourage);

function encourage() {
  // Input 
  let name = nameInEl.value;
  let timeOfDay = timeOfDayEl.value;

  // Output
  alert("Hello " + name);
  alert("Have a great " + timeOfDay + "!");
}

Take Note:

  • be sure to use an event listener to give the user time to type something into the input element!
  • the value of the input element is always a String data type. Convert to a Number data type if required.
