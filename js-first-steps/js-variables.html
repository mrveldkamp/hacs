JavaScript Variables Topic Explanation

Storing Data with Variables

Variables are used to store and manage program data.

A variable is a "named storage container" for data.

To create a variable use the let keyword.


// Declare a Variable: tell JS to make a storage container called age
let age;

Use the assignment operator, =, to put data into the variable.


// Assign data to a Variable: put data into age container
age = 15;

The number 15 is now saved into the memory area associated with the variable named age.

Use the variable name to access the data stored in memory.


// Use Variable: log the contents of the age variable to the console
console.log(age); // Output => 15
console.log(age + 1); // Output => 16

An existing variable may be reassigned to store different data.


// Reassign Variable: put new data into container
age = 18;
console.log(age); // Output => 18

To be concise, we can combine the variable declaration and assignment into a single line.


let age = 15;

Naming Variables

Take note of some rules and conventions to follow when naming a variable:

  • Begin variable names with a lower-case letter
  • Only use letters (a-z, A-Z), numbers (0-9) and underscores (_)
  • Use camel casing for multi-word variables (eg. myCamelCaseVariable)
  • Make variable names intuitive: describe what they store
  • Avoid using JavaScript reserved words (eg. let, alert, console, ...)
