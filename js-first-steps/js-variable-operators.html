JavaScript Variable Operators Topic Explanation

Modify and Assign

It is common to apply an operator to a variable and store the new result in that same variable.


// Declare & Assign
let n = 1;

// Increase n by 5
n = n + 5;

// Decrease n by 3
n = n - 3;

// Multiply n by 2
n = n * 2;

// Divide n by 3
n = n / 3;

// Declare & Assign
let greeting = "Hi";

// Append "!" to greeting
greeting = greeting + "!";

Shorthand Operators

Shorthand operators exist to concisely accomplish the same "modify and assign" tasks as above.


// Declare & Assign
let n = 1;

// Increase n by 5
n += 5; // Same as n = n + 5;

// Decrease n by 3
n -= 3; // Same as n = n - 3;

// Multiply n by 2
n *= 2; // Same as n = n * 2;

// Divide n by 3
n /= 3; // Same as n = n / 3;

let greeting = "Hi";

// Append "!" to greeting
greeting += "!"; // Same as greeting = greeting + "!";

Increment and Decrement

There are special operators for increasing or decreasing a variable by one.


// Declare & Assign
let counter = 1;

// INCREMENT: increase variable by 1
counter++; // Same as counter = counter + 1 or counter += 1
console.log(counter); // Output => 2

// DECREMENT: decrease variable by 1
counter--; // Same as counter = counter - 1 or counter -= 1
console.log(counter); // Output => 1
