JavaScript Number Data Topic Explanation

Number Data Type

Another common type of data is the Number data type.

A Number is simply a numerical value (with or without a decimal) that is not enclosed by quotes.


25
3.14159
42

Number Operations

JavaScript supports common math operations on Numbers.

Addition (+)


console.log(2 + 3); // Output=> 5

Subraction (-)


console.log(7 - 3); // Output=> 4

Multiplication (*)


console.log(6 * 5); // Output=> 30

Division (/)


console.log(3 / 2); // Output=> 1.5

Remainder (%)


console.log(3 % 2); // Output=> 1
console.log(11 % 3); // Output=> 2
console.log(12 % 4); // Output=> 0

Exponent (**)


console.log(5 ** 2); // Output=> 25
console.log(2 ** 3); // Output=> 8
console.log(16 ** 0.5); // Output=> 4

Order of Operations

JavaScript follows proper order of operations and ( ) may be used in calculations to influence the order of operations.


console.log(1 + 2 * 3); // Output=> 7
console.log((1 + 2) * 3); // Output=> 9
