Concept Notes

Random Array Element

Get a random array element by generating a random index value.

Examples

The array below has a length of 6, which means that the index values go from 0 to 5.


let colors = ["red", "green", "blue", "cyan", "yellow", "magenta"];
// (index) ->   0       1        2       3        4         5

Get a random index by generating a random integer between 0 and 5, inclusive.


let randIndex = Math.floor(Math.random() * 6);
console.log(randIndex);
console.log("Random Color: " + colors[randIndex]);

The array below has a length of 4, which means that the index values go from 0 to 3.


let directions = ["N", "E", "S", "W"];
//    (index) ->   0    1    2    3

Get a random index by generating a random integer between 0 and 3, inclusive.


let randIndex = Math.floor(Math.random() * 4);
console.log(randIndex);
console.log("Random Direction: " + directions[randIndex]);

General Case

The index of the last element of an array is always one less than the length of the array.

Get a random value from any array by generating a random integer between 0 and (array.length - 1), inclusive


let myArray = ["?", "?", ... , "?"];
// (index) ->   0    1  ... (length - 1)

let randIndex = Math.floor(Math.random() * myArray.length);
console.log(randIndex);
console.log("Random Element: " + myArray[randIndex]);
