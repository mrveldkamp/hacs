Concept Notes

Traverse Array

To traverse an array means to travel over all the elements in an array.

A loop may be used to generate all of the index values of an array and then those index values can be used to access each of the array elements.

Example


// NO LOOP - Print out all array elements 
let fibonacci = [1, 2, 3, 5, 8, 13, 21];
//   (index) ->  0  1  2  3  4  5   6
console.log(fibonacci[0]);
console.log(fibonacci[1]);
console.log(fibonacci[2]);
console.log(fibonacci[3]);
console.log(fibonacci[4]);
console.log(fibonacci[5]);
console.log(fibonacci[6]);

// WITH LOOP - Print out all array elements
let fibonacci = [1, 2, 3, 5, 8, 13, 21];
//   (index) ->  0  1  2  3  4  5   6
for (let i = 0; i < 7; i++) {
  console.log(fibonacci[i]);
}

General Case

Using the length of the array will make the for loop traverse an array of any size.


// GENERAL CASE - Traverse all elements 
let myArray = ["?", "?", ... , "?"];
for (let i = 0; i < myArray.length; i++) {
  console.log(myArray[i]);
}
