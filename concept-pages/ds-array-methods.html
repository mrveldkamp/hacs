Concept Notes

Array Methods+

JavaScript has some built-in tools that help when working with arrays.

includes() Method

The includes() method determines whether an array contains a specified item.

This method returns true if the array contains the item, and false if not.

array.includes(item); 

let attendanceList = ["Angela", "Bryan", "Carol", "David"];
if (attendanceList.includes("Joe")) {
  console.log("I'm so glad Joe could make it!");
} else {
  console.log("I wonder where Joe is?");
}

indexOf() Method

The indexOf() method searches the array for a specified item.

If the item is found, the method will return the index of the first occurence of the item.

If the item is not found, the method will return -1.

array.indexOf(item); 

let attendanceList = ["Angela", "Bryan", "Carol", "David"];
let index = attendanceList.indexOf("Carol");
if (index === -1) {
  console.log("Carol not found.");
} else {
  console.log(`Carol found at position: ${index}.`);
}

*Note: there is also a lastIndexOf() method that returns the index of the last occurance of the item.

concat() Method

The concat() method merges two arrays into a new combined array.

array.concat(anArray); 

let group1 = ["Angela", "Bryan", "Carol"];
let group2 = ["David", "Ezra"];
let combinedGroup = group1.concat(group2);
// combinedGroup: ["Angela", "Bryan", "Carol", "David", "Ezra"]

reverse() Method

The reverse() method reverses the order of the elements in the array.

array.reverse(); 

let nums = [1, 2, 3, 4, 5];
nums.reverse();
// nums: [5, 4, 3, 2, 1]

Math.max() & Math.min() Methods

The Math.max() and Math.min() methods take in numbers and return the maximum or minimum number.

These methods are designed to work with multiple arguments and not an array argument.

In order to use these methods with arrays, the spread operator (...) may be used to expand the array in place of the required arguments.

Math.max(n1, n2, n3, ..., nX)
Math.min(n1, n2, n3, ..., nX) 

// MAX OF A LIST OF ARGUMENTS
let maxNum = Math.max(50, 20, 30, 70, 40); 
// maxNum: 70

// MAX & MIN OF AN ARRAY
let numArray = [2, 8, 12, 5, 9, 20, 17, 3];

// The Wrong Way
let wrongMaxWay = Math.max(numArray);
// wrongMaxWay: NaN

// The Right Way (...Spread Operator)
let maxElement = Math.max(...numArray);
// maxElement: 20
let minElement = Math.min(...numArray);
// minElement: 2
