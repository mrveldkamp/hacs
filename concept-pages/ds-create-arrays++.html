Create Arrays++

Arrays may be created and initialized in a variety of ways.

Create an Array with a Loop

Loops may be used to create arrays.

Start with an empty array and then add items one at a time inside of a loop.


// Create an array with identical values (100 exclamation marks)
let identicalArray = [];
for (let n = 0; n < 100; n++) {
    identicalArray.push("!");
}

// Create a random array (500 random decimals b/t 0 and 1)
let randomArray = [];
for (let n = 0; n < 500; n++) {
    randomArray.push(Math.random());
}

// Create an array from user input (5 colors)
let inputArray = [];
for (let n = 0; n < 5; n++) {
    inputArray.push(prompt("Please enter a color:"));
}

// Create a patterned array (even #'s b/t 0 and 50, inclusive)
let patternArray = [];
for (let num = 0; num <= 50; num += 2) {
    patternArray.push(num);
}

Create an Array From a String

A string may be turned into an array with the split() method.

The split() method takes in an argument that will be used as the token to separate the string. 


// Create arrays from strings with split()
let namesString = "adam,ben,caleb,david";
let namesArray = namesString.split(",");
// -> ["adam", "ben", "caleb", "david"]

let scoresString = "80;90;75;88;95;99;85";
let scoresArray = scoresString.split(";");
// -> ["80", "90", "75", "88", "95", "99", "85"]

let sentence = "I love arrays";
let wordArray = sentence.split(" ");
// -> ["I", "love", "arrays"]
let characterArray = sentence.split("");
// -> ["I"," ","l","o","v","e"," ","a","r","r","a","y","s"]

The join() method may be used to turn an array back into a string.

The join() method takes in an argument that will be used as the token to place between array elements.


// Convert an array into a string with join()
let wordArray = ["I", "love", "arrays"];

let sentence = wordArray.join(" ");
// -> "I love arrays"

let sentence2 = wordArray.join("-");
// -> "I-love-arrays"

Create an Array of HTML Elements

HTML elements may be selected using querySelectorAll().

The querySelectorAll() method will take any valid css selector and all elements that match the css selector will be returned in a NodeList (which is similar to an array).


// Select all elements with the class "special"
let specialEls = document.querySelectorAll(".special");

// Add an event listener to all elements with the class special
for (let i = 0; i < specialEls.length; i++) {
  specialEls[i].addEventListener("click", doSomethingSpecial);
}
