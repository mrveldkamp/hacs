Concept Notes

Intro to Data Structures

A data structure is a structure for storing data.

Some common data structures are: arrays, objects (associative arrays), arrays of arrays and arrays of objects.

Arrays

An array is an ordered collection of values.


let vowels = ["a", "e", "i", "o", "u"];

Objects (Associative Arrays)

An object (Associative Array) is a collection of property-value pairs.


// One line definition of object
let point = { x: 2, y: 3, label: "A"};

// Same thing but on multiple lines to more clearly see individual properties
let point = {
  x: 2,
  y: 3,
  label: "A"
};

Arrays of Arrays

An array of arrays is useful for grid-based programs with rows and columns.


// One line definition of grid
let grid = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];

// Same thing but on multiple lines to more clearly see grid structure.
let grid = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9] 
];

Arrays of Objects

An array of objects is a common data structure for storing lists of associated data.


// Array of Point Objects
let points = [
  {
    x: 2,
    y: 3,
    label: "A"
  },
  {
    x: -4,
    y: 0,
    label: "B"
  },
  {
    x: 8,
    y: 5,
    label: "C"
  } 
];
