Concept Notes

Intro to Data Structures

A data structure is a structure for storing data.

Some common data structures are:

  • Arrays
  • Parallel Arrays
  • Associative Arrays
  • Arrays of Objects
  • Arrays of Arrays

Arrays

An Array is an ordered collection of values.


let vowels = ["a", "e", "i", "o", "u"];

Parallel Arrays

Parallel Arrays use multiple arrays to store related data.

All the related data for an 'item' will be found at the same position of each array.


// labels and (x, y) values of three points
let pLabels = ["A", "B", "C"];
let xValues = [100, 200, 300];
let yValues = [200, 350, 500];

Point "A": (100, 200) | Point "B": (200, 350) | Point "C": (300, 500)

Associative Arrays (Objects)

An Associative Array is a collection of property-value pairs.

In JavaScript, Associative Arrays are commonly referred to as Objects.


// One line definition of object
let point = { x: 2, y: 3, label: "A"};

// Same thing but on multiple lines to more clearly see individual properties
let point = {
  x: 2,
  y: 3,
  label: "A"
};

Arrays of Objects

An Array of Objects is a common data structure for storing lists of related data.


// Array of Point Objects
let points = [
  {
    x: 2,
    y: 3,
    label: "A"
  },
  {
    x: -4,
    y: 0,
    label: "B"
  },
  {
    x: 8,
    y: 5,
    label: "C"
  } 
];

Arrays of Arrays

An Array of Arrays is useful for grid-based programs with rows and columns.


// One line definition of grid
let grid = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];

// Same thing but on multiple lines to more clearly see grid structure.
let grid = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9] 
];
