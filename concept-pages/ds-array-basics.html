Concept Notes

Array Basics

An array is an ordered collection of values.


let vowels = ["a", "e", "i", "o", "u"];

Create an Array with [ ]

An array can be created by putting comma-separated values into square brackets.


let emptyArray = [];
let numberArray = [1, 2, 3.0, 4, 5];
let stringArray = ["red", "green", "blue"];
let mixedArray = ["text", 7, true, 3.14, [5, 10, 15]];

Access Elements with [ ]

Each element in an array has a position / index.

Index values start at 0.


let letterArray = ["a", "b", "c", "d"];
//     (index) ->   0    1    2    3

The value of an individual array element may be accessed using the index operator: [ ].


let letterArray = ["a", "b", "c", "d"];
console.log("1st element at index 0 is: " + letterArray[0]);
console.log("2nd element at index 1 is: " + letterArray[1]);
console.log("3rd element at index 2 is: " + letterArray[2]);
console.log("4th element at index 3 is: " + letterArray[3]);

Length of an Array

The length of an array may be determined using the length property.


let numArray = [1, 2, 3, 4, 5, 6, 7, 8];
console.log("Length of numArray is: " + numArray.length);
