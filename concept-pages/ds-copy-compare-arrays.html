Copy & Compare Arrays

Copy Arrays

The slice() method may be used to copy all or part of an array.

The slice() method takes in optional start (inclusive) and stop (exclusive) arguments that control the section of the array to copy. The default start value is 0 and the default stop value is the length of the array.


// Array Slice Method 
let anArray = [10, 20, 30, 40, 50, 60, 70];
//   index ->   0   1   2   3   4   5   6

let slice1 = anArray.slice(1, 4); // index 1 to 3: [20, 30, 40] 
let slice2 = anArray.slice(3); // index 3 to end: [40, 50, 60, 70] 
let slice3 = anArray.slice(); // entire array: [10, 20, 30, 40, 50, 60, 70]

Compare Arrays

To compare arrays, one technique is to convert the arrays to strings and then check the strings for equality.

The JSON.stringify() method may be used to convert an array to a string.


// Array Compare with JSON.stringify 
let array1 = [1, 2, 3];
let array2 = [1, 2, 3];
let array3 = [3, 2, 1];

console.log(array1 === array2); 
// -> False (Different memory addresses)

console.log(JSON.stringify(array1) === JSON.stringify(array2)); 
// -> True ("[1,2,3]" === "[1,2,3]")

console.log(JSON.stringify(array1) === JSON.stringify(array3)); 
// False ("[1,2,3]" !== "[3,2,1]")
