Add/Remove Array Elements

There are a number of useful JavaScript array methods that help to add and remove array elements.

array.push() / array.pop()

The push method will add values to the end of an array.


// Add item(s) to the end of an array 
anArray.push(item1, item2, ...);

// Example
let vowels = ["a", "e", "i", "o", "u"];
vowels.push("y"); // ["a", "e", "i", "o", "u", "y"]

The pop method will remove the last element from an array


// Remove last element
anArray.pop();

// Remove and store last element
let lastElement = anArray.pop();

// Example
let vowels = ["a", "e", "i", "o", "u", "y"];
vowels.pop(); // ["a", "e", "i", "o", "u"]

array.splice()

The splice method may add/or remove array elements at specific indeX.


// Add and/or Remove array elements at a specific index 
anArray.splice(index, howManyToRemove, item1, item2, ...);

// Example - remove 4th element 
let colors = ["red", "green", "blue", "yellow", "cyan"];
colors.splice(3, 1); // ["red", "green", "blue", "cyan"]

// Example - insert a value at index 2
let colors = ["red", "green", "blue"];
colors.splice(2, 0, "magenta"); // ["red", "green", "magenta", "blue"]

// Example - remove 1st element and insert a value
let colors = ["red", "green", "blue"];
colors.splice(0, 1, "magenta"); // ["magenta", "green", "blue"]
