innerHTML

Topic Explanation

JavaScript is able to set the content of an HTML element with the innerHTML property.

The innerHTML of an element is the content between the element's opening and closing tags.

index.html


<body>
<h1 id="heading">This is a Heading</h1>
<p id="para">This is a paragraph.</p>
<p><button id="btn">Click Me</button></p>
  
<!-- Run External JavaScript File -->
<script src="script.js"></script>
</body>

script.js

The code below will change the content of the heading and paragraph elements when the button element is clicked. 


// Variables to store references to HTML elements
let btnEl = document.getElementById("btn");
let headingEl = document.getElementById("heading");
let paraEl = document.getElementById("para");

// Run changeContent function when button is clicked 
btnEl.addEventListener("click", changeContent);

// Use innerHTML to change content of heading and paragraph
function changeContent() {
  headingEl.innerHTML = "New Heading";
  paraEl.innerHTML = "New <em>Awesome</em> Paragraph.";
}
