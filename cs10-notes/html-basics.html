Intro to HTML

HTML stands for HyperText Markup Language.

HTML tells web browsers the content and structure of a web page.

Below is an example of some HTML code and how a web browser displays the code.

HTML Code (index.html)


<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Main Heading</h1>
    <p>
      Lorem ipsum dolor sit amet consectetur adipisicing elit. Eveniet,
      veritatis repellat modi dolorem officiis accusantium.
    </p>

    <h2>Sub Heading</h2>
    <p>
      <img src="https://picsum.photos/id/0/400/250" alt="Laptop" />
    </p>
    <p>
      Lorem ipsum dolor, sit amet consectetur adipisicing elit. Quaerat ullam
      tempora <a href="https://picsum.photos/">perferendis</a> impedit.
    </p>
    <ul>
      <li>List item 1</li>
      <li>List item 2</li>
      <li>List item 3</li>
    </ul>
  </body>
</html>

Web Browser Display of above HTML Code

HTML Code is written in a text editor and saved to a file with a ".html" extension.

The home page of a web site should be named: index.html.

HTML Elements

HTML uses html elements to describe the content of a web page.

Common HTML elements are:

  • heading elements: <h1>, <h2>, ... ,<h6>
  • paragraph elements: <p>
  • image elements: <img>
  • link elements: <a>
  • list elements: <ol>, <ul>, <li>

Basic HTML Elements

Basic HTML elements consist of an opening tag, content and a closing tag.

HTML Attributes

HTML elements may have attributes that provide additional information about the element.

Attributes go in the opening tag and usually come in name/value pairs: name="value".

A link element uses an href attribute to specify where the link should take you when clicked. 


<a href="https://www.google.ca/">Visit Google</a>

Void HTML Elements

Void HTML elements consist of a single tag and do not follow the pattern of an opening tag, content and a closing tag.

Void elements are typically used to insert/embed something in the web page.

An image element is an example of a void element.


<img src="picture.jpg" />

The image element doesn't have a closing tag or any content. Instead it uses a src attribute to tell the browser the source of the image to embed in the web page.

Anatomy of an HTML Document

HTML Page Structure

A simple html document has the following structure:


<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h2>Level 2 Heading</h2>
    <p>A paragraph.</p>
  </body>
</html>
  • <!DOCTYPE html> declares that this is an HTML5 document.
  • The html element is the root element of the HTML page. This element wraps all the content of the entire page.
  • The head element contains metadata (descriptive informaiton) about the HTML page.
  • The title element specifies a title for the HTML page that is used in the web browser's tab.
  • The body element is a container for all of the visible content of the HTML page.

HTML DOM

HTML code uses a Document Object Model (DOM) to structure HTML elements in parent, child and sibling relationships.

  • Child elements are nested inside parent elements.
  • Sibling elements have the same parent.

The HTML Dom for the Page Structure code above.

Some examples of parent, child, sibling relationships from the HTML Dom above:

  • The title element is a child of the head element
  • The h2 and p elements are children of the body element.
  • The h2 and p elements are sibling.
