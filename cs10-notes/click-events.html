Document Event Listener

An Event Listener is used to run code when a certain event occurs.

The following code will add a Click Event Listener to the HTML document. Every time the document is clicked, the message "Hello Document!" will be displayed in an alert-box.

main.js


// Add Click Event Listener to HTML document
document.addEventListener("click", greeting);

function greeting() {
  alert("Hello Document!");
}

The greeting function stores code that will only run when the function is invoked/called.

When we add a Click Event Listener to the document, we give the Event Listener the name of a function to invoke/call when the provided event occurs. In this case, the Event Listener will call the function named greeting when the "click" event occurs.

Element Event Listener

Event Listeners may also be added to HTML elements.

The following code will add a Click Event Listener to an HTML button element. Every time the button is clicked, the message "Hello Button!" will be displayed in an alert-box and in the console.

index.html


<body>
  <h1 id="heading">This is a Heading</h1>
  <p><button id="btn">Click Me</button></p>
</body>

main.js


// Run greetButton function when button is clicked 
document.getElementById("btn").addEventListener("click", greetButton);

function greetButton() {
  console.log("Hello Button!");
  alert("Hello Button!");
}

Summary

  • Functions may be used to store blocks of code.
  • Event listeners may be added to the document or specific HTML elements. When the specified event occurs, the event listener will tell the provided function to run its code.
