String Data Type

A String is a sequence of characters used to represent text.

Strings are enclosed by matching double quotes, single quotes or backticks. 


let string1 = "doubled-quoted String data";
let string2 = 'single-quoted String data';
let string3 = `backtick-quoted String data`;

The backtick key is located above the left Tab key on most keyboards.

String Concatenation

The operation of joining strings together is called concatenation.

In JavaScript, concatenation is done with the + operator. 


"Hello" + "World"; // Result => "HelloWorld"
"Hi" + " There!"; // Result => "Hi There!"

String Properties & Methods

There are some useful properties and methods that work on String data.

Length Property

The length property of a string will return the number of characters in a string. 


"hello".length; // Result => 5
"Hi There".length; // Result => 8

toLowerCase() / toUpperCase() Methods

The toLowerCase() and toUpperCase() methods will convert a string to either all lowercase or all uppercase letters. 


"Algorithm".toLowerCase(); // Result => "algorithm"
"JavaScript".toUpperCase(); // Result => "JAVASCRIPT"

Template Strings

When a string is enclosed with backticks, we can embed variables into the string by wrapping the variables in ${...}. 


let name = "John";
let timeOfDay = "Morning";
let greeting = `Hello ${name}. Have a great ${timeOfDay}!`; 
// greeting  => "Hello John. Have a great Morning!"

A String made with backticks is commonly called a Template String.

Summary

  • The String data type is used to represent text.
  • Join strings together (concatenate) with the + operator.
  • Use the length property to get the number of characters in a string.
  • Use the toLowerCase() and toUpperCase() methods to convert strings to all lowercase or all uppercase.
  • A Template String is made using backticks. Variables may be inserted into a Template String with ${...}.
