Math Object Basics

JavaScript has a built in Math object which contains a small library of mathematical functions and constants.

Here are a few examples:

Math.random()


// Generate a random decimal between 0 and 1
Math.random(); // Possible Result => 0.2743923

Math.sqrt()


// Determine the square root of a provided number
Math.sqrt(64); // Result => 8

Math.floor()


// Round a number DOWN to the nearest integer
Math.floor(5.892); // Result => 5
Math.floor(5.265); // Result => 5

Math.ceil()


// Round a number UP to the nearest integer
Math.floor(5.892); // Result => 6
Math.floor(5.265); // Result => 6

Math.round()


// Round a number to the nearest integer
Math.round(5.892); // Result => 6
Math.round(5.265); // Result => 5

Math.PI


// Use Math.PI for an approximate value of pi (Π)
Math.PI; // Result -> 3.141592653589793
Math.PI * 5 ** 2; // Result => 78.53981633974483

More examples can be found at the MDN Web Docs.
