Input-Process-Output Algorithm

A common algorithm is an Input, Process, Output algorithm.

This algorithm involves:

  1. Input - get input from the user.
  2. Process - process the input in some way (ie. calculations).
  3. Output - output the result.

// SIMPLE GREETING (IPO Algorithm)

// Input - ask user for name and time of day
let name = prompt("Please enter your name:");
let timeOfDay = prompt("Please enter the time of day:");

// Process - use inputs to create greeting
let greeting = `Hello ${name}. Have a great ${timeOfDay}!`;

// Output - display greeting
alert(greeting);
  

// SIMPLE ADDITION CALCULATOR (IPO Algorithm)

// Input - ask user for two numbers
let num1 = +prompt('Please enter a number:'');
let num2 = +prompt('Please enter another number:');

// Process - calculate sum
let total = num1 + num2;

// Output - display result
alert(`${num1} + ${num2} is ${total}`);
