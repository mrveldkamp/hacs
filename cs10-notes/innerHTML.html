Set Element Content

JavaScript is able to access the content of an HTML element with the innerHTML property.

The innerHTML of an element is the content between the element's opening and closing tags.

index.html


<!DOCTYPE html>
<html>
<head>
  ...
  <script defer src="main.js"></script>
</head>

<body>
  <h1 id="heading">This is a Heading</h1>
  <p id="para">This is a paragraph.</p>
  <p><button id="btn">Click Me</button></p>
  ...
</body>
</html>

The code below will use the innerHTML property to change the content of the heading and paragraph elements when the button element is clicked.

main.js


// Run changeContent function when button is clicked 
document.getElementById("btn").addEventListener("click", changeContent);

// Use innerHTML to change content of heading and paragraph
function changeContent() {
  document.getElementById("heading").innerHTML = "New Heading";
  document.getElementById("para").innerHTML = "New <em>Awesome</em> Paragraph.";
}

Summary

  • Use an elements innerHTML property to set the content of the element.
