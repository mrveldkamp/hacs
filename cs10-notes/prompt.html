Prompt()

JS Notes Video Notes

Text Input

The prompt() function will open a prompt-box where the user can provide text input. The text input that is returned from the prompt() function may be stored in a variable for later use. 


// Ask user for their name and greet them by name
let name = prompt("Please enter your name:");
alert(`Hello ${name}!`);

Numerical Input

Input received from the prompt() function is always returned as a String data type.

If you want numerical input, convert the String that prompt() returns to a Number data type using the unary + operator or the Number() function. 


// Convert user input to Number data type with '+' operator
let num1 = +prompt("Please enter a number:");

// Convert user input to Number data type with Number() function
let num2 = Number(prompt("Please enter a number:"));

Summary

  • The prompt() function may be used to get input from the user via a prompt-box.
  • The prompt() function always returns user input as a String data type.
  • Use the unary + operator or the Number() function to convert String input to a Number.
