While Loops

JS Notes Video Notes

A while loop is a type of repetition control structure.

While Loop Basics

The code inside of a while loop will repeatedly execute as long as the condition is true. 


while (condition) {
  // Code block 
  // to be 
  // executed...
}

Example: Log the numbers 1 to 100 to the console...


let num = 1;
while (num <= 100) {
  console.log(num);
  num++;
}

There are 4 key parts of executing a while loop:

  1. Initialize a variable (let num = 1)
  2. Test a condition (num <= 100)
  3. Loop body (console.log(num))
  4. Update variable (num++)

Warning - Infinite Loops

An infinite loop is a loop that continuously executes and will never end.

Common causes of infinite loops include:

  • A faulty condition that will always be true.
  • Forgetting to update the variable that is tested in the condition.

Summary

  • A while loop will repeatedly execute the code inside of the loop body as long as the provided condition is true.
  • Be careful of infinite loops.
