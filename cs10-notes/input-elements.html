HTML <input> Element

An <input> element will display as an input-box on a web page: .

An <input> element may have id and value attributes.

  • id attribute: use getElementById() to get a reference to the element.
  • value attribute: may be used to initialize the value of the input element.

index.html


<p>Enter your name: <input id="name-in"></p>
<p>Enter the time of day: <input id="time-in" value="Morning"></p>
<p><button id="btn">Submit</button></p>
<p id="output">-----</p>

Output of index.html

Enter your name:

Enter the time of day:

-----

Get Value of <input> Element

To get the value of an <input> element: use getElementById() to get a reference to the <input> element and then access the value property of the <input> element to get the contents.

main.js


// Add Event Listener to button element
document.getElementById("btn").addEventListener("click", encourage);

function encourage() {
  // Input 
  let name = document.getElementById("name-in").value;
  let timeOfDay =document.getElementById("time-in").value;

  // Process
  let msg = `Hello ${name}. Have a great ${timeOfDay}!`;

  // Output
  document.getElementById("output").innerHTML = msg;
}

Attention

  • be sure to use an event listener to give the user time to type something into the input element!
  • the value of the input element is always a String data type. Convert to a Number data type if required.

Summary

  • JavaScript can get the value of <input> elements.
  • The value of an <input> element is always a String.
