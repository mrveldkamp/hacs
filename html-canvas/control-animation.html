Control Animation

Video Lesson

If statements provide the ability to control animations.

Animate if True | Reset Animation

Animate If True

Put the code that changes the animation variable into an if statement. This will make the animation run only when the if statement condition is true.


let x = 100;

requestAnimationFrame(draw);
function draw() {
  // Update Variable if x < 300
  if (x < 300) {
    x += 2;
  } 

  // Draw Background 
  ctx.fillStyle = "white";
  ctx.fillRect(0, 0, cnv.width, cnv.height);

  // Draw Rectangle (using x)
  ctx.fillStyle = "green";
  ctx.fillRect(x, 100, 50, 50);

  // Run draw again... 
  requestAnimationFrame(draw);
}

Reset Animation

An if statement may be used to test the state of an animation variable and then reset the value of that variable. 


let x = 100;

requestAnimationFrame(draw);
function draw() {
  // Update Variable
  x += 2;

  // Reset Animation if x > 300
  if (x > 300) {
    x = 100;
  }

  // Draw Background 
  ctx.fillStyle = "white";
  ctx.fillRect(0, 0, cnv.width, cnv.height);

  // Draw Rectangle (using x)
  ctx.fillStyle = "green";
  ctx.fillRect(x, 100, 50, 50);

  // Run draw again... 
  requestAnimationFrame(draw);
}

Summary

  • If statements can make an animation run while a condition is true.
  • Reset an animation by testing the state of the animation.
