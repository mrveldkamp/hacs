requestAnimationFrame()

JavaScript has a requestAnimationFrame() function that is designed to help with canvas animations.

  • Put the code that you want to repeat inside of a function (Let's call it 'draw').
  • Use the requestAnimationFrame() function to invoke the 'draw' function.
  • Use the requestAnimationFrame() function again at the end of the 'draw' function to invoke the 'draw' function again. This will create a cycle of the draw function continually invoking itself. 

requestAnimationFrame(draw);
function draw() {
  // Code to repeat 
  console.log("hello");

  // Run draw again... 
  requestAnimationFrame(draw);
}

The requestAnimationFrame() function requests the browser to determine the optimal FPS (frames per second) for the animation: usually around 60 frames per second.

Variables & Animation

Variables play an important part in making a canvas animation.

Variables are used to change the picture that is drawn.


let x = 100;

requestAnimationFrame(draw);
function draw() {
  // Update Variable 
  x += 2;

  // Draw Background 
  ctx.fillStyle = "white";
  ctx.fillRect(0, 0, cnv.width, cnv.height);

  // Draw Rectangle (using x)
  ctx.fillStyle = "green";
  ctx.fillRect(x, 100, 50, 50);

  // Run draw again... 
  requestAnimationFrame(draw);
}

By changing the variable x, the position of the rectangle will change each time the canvas is redrawn.

Attention

  • use variables to animate the location, size, color, ... of items drawn on the canvas.

Summary

  • requestAnimationFrame() is used to consistently redraw the canvas.
  • Variables are used to change the picture that is drawn.
