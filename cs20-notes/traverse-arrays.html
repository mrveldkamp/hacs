Access Each Element in an Array

To traverse an array means to travel over all the elements in an array.

A loop may be used to generate all of the index values of an array. Then those index values can be used to access each of the array elements. 


let fibonacci = [1, 2, 3, 5, 8, 13, 21];
//   (index) ->  0  1  2  3  4  5   6

// WITHOUT A LOOP - Print out all array elements 
console.log(fibonacci[0]);
console.log(fibonacci[1]);
console.log(fibonacci[2]);
console.log(fibonacci[3]);
console.log(fibonacci[4]);
console.log(fibonacci[5]);
console.log(fibonacci[6]);

// WITH A LOOP - Print out all array elements
for (let i = 0; i < 7; i++) {
  console.log(fibonacci[i]);
}

Traverse - General Case

The length of an array may be determined using the length property.


let vowels = ['a', 'e', 'i', 'o', 'u'];
//   index ->  0    1    2    3    4 
vowels.length; // -> 5

Using the length of the array will allow the for loop to traverse an array of any size. 


// GENERAL CASE - Traverse all elements 
let myArray = ["?", "?", ... , "?"];
//   index ->   0    1 ... (length - 1)

// Traverse array - use index 0 up to 
// but not including length of array
for (let i = 0; i < myArray.length; i++) {
  console.log(myArray[i]);
}
