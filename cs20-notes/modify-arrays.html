Modify Elements with [ ]

You may reassign / modify array elements when accesing the element with [ ]. 


let nums = [1, 2, 3, 4, 5, 6];
// index -> 0  1  2  3  4  5

nums[0] = 7;  // [7, 2, 3, 4, 5, 6]
nums[3] += 5; // [7, 2, 3, 9, 5, 6]
nums[5]++;    // [7, 2, 3, 9, 5, 7]

array.push() / array.pop()

The push method will add values to the end of an array. 


// Add value to end of array
let vowels = ["a", "e", "i", "o", "u"];
vowels.push("y"); // ["a", "e", "i", "o", "u", "y"]

The pop method will remove the last element from an array 


// Remove last element
let vowels = ["a", "e", "i", "o", "u", "y"];
vowels.pop(); // ["a", "e", "i", "o", "u"]

// The pop() method also returns the last element
let lastElement = anArray.pop();

array.splice()

The splice method may add/or remove array elements at a specific index. 


// Add and/or Remove array elements at a specific index 
anArray.splice(index, howManyToRemove, itemToAdd1, itemToAdd2, ...);
  

// Remove 4th element 
let colors = ["red", "green", "blue", "yellow", "cyan"];
colors.splice(3, 1); // ["red", "green", "blue", "cyan"]
  

// Insert a value at index 2
let colors = ["red", "green", "blue"];
colors.splice(2, 0, "magenta"); // ["red", "green", "magenta", "blue"]
