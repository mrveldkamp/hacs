Create an Array with [ ]

An array can be created by putting comma-separated values into square brackets. 


let emptyArray = [];
let numberArray = [1, 2, 3.0, 4, 5];
let stringArray = ["red", "green", "blue"];
let mixedArray = ["text", 7, true, 3.14, [5, 10, 15]];

Create an Array From a String

A string may be turned into an array with the split() method.

The split() method takes in an argument that will be used as the token to separate the string. 


// CREATE ARRAYS FROM STRINGS WITH SPLIT()
let namesString = "adam,ben,caleb,david";
let namesArray = namesString.split(",");
// -> ["adam", "ben", "caleb", "david"]

let scoresString = "80;90;75;88;95;99;85";
let scoresArray = scoresString.split(";");
// -> ["80", "90", "75", "88", "95", "99", "85"]

let sentence = "I love arrays";
let wordArray = sentence.split(" ");
// -> ["I", "love", "arrays"]
let characterArray = sentence.split("");
// -> ["I"," ","l","o","v","e"," ","a","r","r","a","y","s"]

The join() method may be used to turn an array back into a string.

The join() method takes in an argument that will be used as the token to place between array elements. 


// Convert an array into a string with join()
let wordArray = ["I", "love", "arrays"];

let sentence = wordArray.join(" ");
// -> "I love arrays"

let sentence2 = wordArray.join("-");
// -> "I-love-arrays"

Create an Array with a Loop

Loops may be used to create arrays.

Start with an empty array and then use a loop to add items one at a time to the end of the array using the push() method. 


// Create an array with identical values (100 exclamation marks)
let identicalArray = [];
for (let n = 0; n < 100; n++) {
  identicalArray.push("!");
}

// Create a random array (500 random decimals b/t 0 and 1)
let randomArray = [];
for (let n = 0; n < 500; n++) {
  randomArray.push(Math.random());
}

// Create an array from user input (5 colors)
let inputArray = [];
for (let n = 0; n < 5; n++) {
  inputArray.push(prompt("Please enter a color:"));
}

// Create a patterned array (even #'s b/t 0 and 50, inclusive)
let patternArray = [];
for (let num = 0; num <= 50; num += 2) {
  patternArray.push(num);
}

Create an Array from a TEXT File

The following code shows how to load data from a text file.

Each line of the text file will become an array element.

names.txt


Arya
Badal
Cameron
David 
Ezra
Fred
Grace
Hannah
...

main.js


// Load data from names.txt
let names;
fetch('names.txt')
  .then((res) => res.text())
  .then((data) => names = data.split(/\r?\n/));

The global variable names will now be an array of names as defined in the names.txt file.

fetch() loads the contents of the file into res.

res.text() then converts the text file content to a string and loads it into data.

Data is then split using a regular expression that catches different possible line endings: /r, /n, /r/n.

Global variable names is then assigned the array that data has been split into.

Create an Array from a JSON File

JSON stands for JavaScript Object Notation.

JSON is a data-interchange format that may be used to convert strings to data structures and vice-versa.

Here we load a file containing a JSON string and convert it into the data structure that the string represents.

points.json


[{
  "x": 2,
  "y": 3,
  "label": "A"
}, {
  "x": -4,
  "y": 0,
  "label": "B"
}, {
  "x": 8,
  "y": 5,
  "label": "C"
}]

main.js


// Load data from points.json
let points;
fetch('points.json')
    .then((res) => res.json())
    .then((data) => points = data);

The global variable points will now be an array of point objects as defined in the points.json file.

fetch() loads the contents of the file into res.

res.json() then converts the JSON string from the file into a data structure and loads it into data.

Global variable points is then assigned data.
