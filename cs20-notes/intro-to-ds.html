Data Structures

A data structure is a structure for storing data.

Some common data structures are:

  • Arrays
  • Objects
  • Arrays of Objects
  • Arrays of Arrays

Intro to Arrays

An Array is an ordered collection of values.


let vowels = ["a", "e", "i", "o", "u"];
//   index ->  0    1    2    3    4

You can look up the values in the array using brackets, [ ], and the position or index of the value you're looking for. 


vowels[0]; // -> "a"
vowels[1]; // -> "e"
vowels[2]; // -> "i"
vowels[3]; // -> "o"
vowels[4]; // -> "u"

The length of an array can be found by using the length property. 


vowels.length; // -> 5

Intro to Objects

An Object is a collection of key-value pairs, called properties.


// One line definition of object
let point = { x: 2, y: 3, label: "A"};

// Same thing but on multiple lines to more 
// clearly see individual properties
let point = {
  x: 2,
  y: 3,
  label: "A"
};

You can look up the values in the object by their key.


// Dot Notation
point.x; // -> 2
point.y; // -> 3
point.label; // -> "A"

// Access with []
point["x"]; // -> 2
point["y"]; // -> 3
point["label"]; // -> "A"

Intro to Arrays of Objects

An Array of Objects is a common data structure for storing lists of related data. 


// Array of Point Objects
let points = [
  {
    x: 2,
    y: 3,
    label: "A"
  },
  {
    x: -4,
    y: 0,
    label: "B"
  },
  {
    x: 8,
    y: 5,
    label: "C"
  } 
];

// Access objects in array by index...
// ... then access object properties.
points[0].label; // -> "A"
points[0].x; // -> 2
points[1].label; // -> "B"
points[2].y; // -> 5

Intro to Arrays of Arrays

An Array of Arrays is useful for grid-based data with rows and columns.


// One line definition of grid
let grid = [[1, 2, 3], [4, 5, 6], [7, 8, 9]];

// Same thing but on multiple lines to 
// more clearly see grid structure.
let grid = [
  [1, 2, 3],
  [4, 5, 6],
  [7, 8, 9] 
];

grid[0][0]; // -> 1
grid[2][0]; // -> 7
grid[1][2]; // -> 6
