Return Data from Function

A function may return a value.

Simply use the return keyword followed by the value to return.

The return command stops execution of the function and returns the value to where the function was called.


// DEFINE A RANDOM DECIMAL FUNCTION
// Return a random decimal between low (inclusive) and high (exclusive)
function randomDec(low, high) {
  return Math.random() * (high - low) + low;
}

// INVOKE FUNCTION AND USE RETURN VALUE IN VARIOUS WAYS
// Store returned value in a variable
let xVal = randomDec(0, 300); 

// Pass returned value into console.log
console.log(randomDec(1, 10)); 

// Use returned value in an expression
let circleArea = Math.PI * randomDec(1, 50) ** 2;
