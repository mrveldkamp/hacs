Search an Array

The simplest way to search for a value in an array is to traverse the array to check every element in the array:

  • Loop through the array and check each element to see if it is the value you're looking for.
  • If the loop ends without finding the value, you can safely say that the value is not in the array.

Average of Array Elements

To find the average of all the elements in an array, add up all the values in the array and then divide by the number of elements in the array.

  • Traverse the array to add up all of the array elements.
  • The number of array elements is the same as the length of the array.

Maximum Array Element

To find the maximum value in an array:

  • Assume that the first element is the maximum value.
  • Loop through the rest of the array and update the maximum value when you find a value larger than the current maximum value.
