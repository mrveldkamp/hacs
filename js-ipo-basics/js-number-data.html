JS Number Data

Number Data Type

A Number is simply a numerical value (with or without a decimal) that is not enclosed by quotes.


25
3.14159
42

Number Operations

JavaScript supports common math operations on Numbers.

Addition (+)


2 + 3; // Result => 5

Subraction (-)


7 - 3; // Result => 4

Multiplication (*)


6 * 5; // Result => 30

Division (/)


3 / 2; // Result => 1.5

Remainder (%)


3 % 2; // Result => 1
11 % 3; // Result => 2
12 % 4; // Result => 0

Exponent (**)


5 ** 2; // Result => 25
2 ** 3; // Result => 8
16 ** 0.5; // Result => 4

Order of Operations

JavaScript follows proper order of operations and ( ) may be used in calculations to influence the order of operations.


1 + 2 * 3; // Result => 7
(1 + 2) * 3; // Result => 9

toFixed() Method

Numbers may be rounded to a specified decimal place using the toFixed() method.

The toFixed() method rounds a number and converts the number to a String data type. 


23.451.toFixed(2); // Result => "23.45"
7.2974.toFixed(1); // Result => "7.3"

This method is often used when outputting the result of a calculation and you want to displaly a rounded value.

Summary

  • The Number data type is used to represent numbers.
  • JavaScript supports the basic math operations for numbers.
  • The toFixed( method rounds a number to a specified number of decimal places and returns the result as a String.
