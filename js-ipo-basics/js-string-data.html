JS String Data

String Data Type

A String is a sequence of characters used to represent text.

Strings are enclosed by matching double or single quotes.


let double = "doubled-quoted String data";
let single = 'single-quoted String data';

String Concatenation

The operation of joining strings together is called concatenation.

In JavaScript, concatenation is done with the + operator.


console.log("Hello" + "World"); // Output=> "HelloWorld"
console.log("Hi" + " There!"); // Output=> "Hi There!"

String Properties & Methods

There are some useful properties and methods that work on String data.

Length Property

The length property of a string will return the number of characters in a string. 


console.log("hello".length); // Output=> 5
console.log("Hi There".length); // Output=> 8

toLowerCase() / toUpperCase() Methods

The toLowerCase() and toUpperCase() methods will convert a string to either all lowercase or all uppercase letters.


console.log("Algorithm".toLowerCase()); // Output=> "algorithm"
console.log("JavaScript".toUpperCase()); // Output=> "JAVASCRIPT"

Template Strings

A String can also be enclosed with backticks. (The key above 'Tab' on most keyboards)


let backtick = `backtick String data`;

Backticks allow us to embed any expression into the string, by wrapping the expression in ${...}. 


let name = "John";
console.log(`Hello ${name}!`); // Output => "Hello John!"

console.log(`2 + 3 = ${2 + 3}`); // Output => 2 + 3 = 5

console.log(`A random number is: ${Math.random()}`);
// Output => "A random number is: 0.73250932"

A String made with backticks is commonly called a Template String.

Summary

  • The String data type is used to represent text.
  • Join strings together (concatenate) with the + operator.
  • Use the length property to get the number of characters in a string.
  • Use the toLowerCase() and toUpperCase() methods to convert strings to all lowercase or all uppercase.
  • A Template String is made using backticks. Expressions may be inserted into a Template String with ${...}.
