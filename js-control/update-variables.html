Update Variables

Before studying loops, it is important to understand how to update variables...

Modify & Assign

It is common to apply an operation to a variable and store the new result in the same variable.


let n = 1;

// Increase n by 5
n = n + 5;

// Decrease n by 3
n = n - 3;

// Multiply n by 2
n = n * 2;

// Divide n by 3
n = n / 3;

let greeting = "Hi";

// Append "!" to greeting
greeting = greeting + "!";

Shorthand Operators

Shorthand operators exist to accomplish the same "modify & assign" tasks as above.


let n = 1;

// Increase n by 5
n += 5; // Same as n = n + 5;

// Decrease n by 3
n -= 3; // Same as n = n - 3;

// Multiply n by 2
n *= 2; // Same as n = n * 2;

// Divide n by 3
n /= 3; // Same as n = n / 3;

let greeting = "Hi";

// Append "!" to greeting
greeting += "!"; // Same as greeting = greeting + "!";

Increment and Decrement

Since it is very common to increase or decrease a variable by one, special operators exist to do this:

  • Increment Operator (increase by 1): ++
  • Decrement Operator (decrease by 1): --


let counter = 1;

// Increment: increase variable by 1
counter++; // Same as counter = counter + 1 or counter += 1
console.log(counter); // Output => 2

// Decrement: decrease variable by 1
counter--; // Same as counter = counter - 1 or counter -= 1
console.log(counter); // Output => 1

Summary

  • It is common to apply an operation to a variable and store the new result in the same variable.
  • Shorthand operators exist that may be used to update variables concisely.
