JS Boolean Expressions

Topic Explanation

Boolean Data Type

JavaScript has a Boolean Data Type that can be one of two possible values: true or false.


let validInput = true;
let found = false;

Boolean Expressions

Mathemetical expression evaluate to a Number data type...

Boolean expressions evaluate to a Boolean data type...

Boolean Expressions are expressions that evaluate to true or false.


let age = 15;
console.log(age < 18); // Output=> true
console.log(age < 10); // Output=> false

Mathematical expressions are made with mathematical operators...

Boolean expressions are made with comparison and logical operators...

Comparison Operators

Comparison operators are used to build boolean expressions:

  • Equal To: ===
  • Not Equal To: !==
  • Greater Than: >
  • Greater Than or Equal To: >=
  • Less Than: <
  • Less Than or Equal To: <=

Examples of Comparison Operators


let age = 15;

// Equal To (===)
console.log(age === 15); // Output=> true
console.log(age === 18); // Output=> false

// Not Equal To (!==)
console.log(age !== 15); // Output=> false
console.log(age !== 13); // Output=> true

// Greater Than (>)
console.log(age > 15); // Output=> false
console.log(age > 10); // Output=> true

// Greater Than or Equal To (>=)
console.log(age >= 15); // Output=> true
console.log(age >= 17); // Output=> false

// Less Than (<)
console.log(age < 20); // Output=> true
console.log(age < 15); // Output=> false

// Less Than or Equal To (<=)
console.log(age <= 20); // Output=> true
console.log(age <= 15); // Output=> true

Logical Operators

Logical Operators allow you to build more complex boolean expressions by connecting simple boolean expressions:

  • Logical And: &&
  • Logical Or: ||
  • Logical Not: !

Examples of Logical Operators


let age = 15;

// And (&&) is true if all connected boolean expressions are true
console.log(age > 10 && age < 20); // Output=> true
console.log(age > 10 && age < 14); // Output=> false

// Or (||) is true if at least one of the connected boolean expressions is true
console.log(age < 10 || age > 20); // Output=> false
console.log(age < 10 || age > 13); // Output=> true
console.log(age > 10 || age < 18); // Output=> true

// Not (!) changes true to false and false to true
let found = true;
console.log(!found); // Output=> false

found = false;
console.log(!found); // Output=> true
