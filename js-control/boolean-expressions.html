JS Boolean Expressions

Boolean Data Type

JavaScript has a Boolean Data Type that can be one of two possible values: true or false.


let validInput = true;
let found = false;

Boolean Expressions

Mathemetical expression evaluate to a Number data type...

Boolean expressions evaluate to a Boolean data type...

Boolean Expressions are expressions that evaluate to true or false.


let age = 15;
age < 18; // Result => true
age < 10; // Result => false

Mathematical expressions are made with mathematical operators (+, -, *, /)...

Boolean expressions are made with comparison operators and logical operators. These operators are explored below...

Comparison Operators

Comparison operators are used to build boolean expressions:

  • Equal To: ===
  • Not Equal To: !==
  • Greater Than: >
  • Greater Than or Equal To: >=
  • Less Than: <
  • Less Than or Equal To: <=

Examples of Comparison Operators


let age = 15;

// Equal To (===)
age === 15; // Result => true
age === 18; // Result => false

// Not Equal To (!==)
age !== 15; // Result => false
age !== 13; // Result => true

// Greater Than (>)
age > 15; // Result => false
age > 10; // Result => true

// Greater Than or Equal To (>=)
age >= 15; // Result => true
age >= 17; // Result => false

// Less Than (<)
age < 20; // Result => true
age < 15; // Result => false

// Less Than or Equal To (<=)
age <= 20; // Result => true
age <= 15; // Result => true

Logical Operators

Logical Operators allow you to build more complex boolean expressions by connecting simple boolean expressions:

  • Logical And: &&
  • Logical Or: ||
  • Logical Not: !

Examples of Logical Operators


let age = 15;

// And (&&) is true if all connected boolean expressions are true
age > 10 && age < 20; // Result => true
age > 10 && age < 14; // Result => false

// Or (||) is true if at least one of the connected boolean expressions is true
age < 10 || age > 20; // Result => false
age < 10 || age > 13; // Result => true
age > 10 || age < 18; // Result => true

// Not (!) changes true to false and false to true
let found = true;
!found; // Result => false

found = false;
!found; // Result => true

Summary

  • Boolean data type is true or false.
  • Boolean expressions evaluate to true or false.
  • Boolean expressions are made up of comparison and logical operators.
