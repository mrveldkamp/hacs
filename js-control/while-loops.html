JS While Loops

Topic Explanation

The code inside of a while loop will repeatedly execute as long as the condition is true.


while (condition) {
  // Code block 
  // to be 
  // executed...
}

Example: Log the numbers 1 to 100 to the console...


let num = 1;
while (num <= 100) {
  console.log(num);
  num++;
}

There are 4 key parts of executing a while loop:

  1. Initialize variable (let num = 1)
  2. Test a condition (num <= 100)
  3. Loop body (console.log(num))
  4. Update variable (num++)
