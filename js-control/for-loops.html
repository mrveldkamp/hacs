JS For Loops

A for loop is a type of repetition control structure.

For Loop Basics

The code inside of a for loop will repeatedly execute as long as the condition is true.


for (init; condition; update) {
  // Code block 
  // to be 
  // executed...
}

For Loop - Flow of Control

  1. execute init statement
  2. if condition is true:
    1. execute loop body
    2. execute update
    3. go to step 2

Example: Log the numbers 1 to 100 to the console...


for (let num = 1; num <= 100; num++) {
  console.log(num);
}

Summary

  • A for loop is a concise way to repeat code.
