Uses of Loops

Loops are commonly used to accomplish the following:

  • Repeat code a set number of times.
  • Repeat code an unknown number of times.

Repeat Set # of Times

For loops often work best when repeating code a set number of times.

In the example below, a for loop is used to simulate flipping a coin 500 times.


// Coin Flip Simulator
let numHeads = 0;
let numTails = 0

// Loop 500 times
for (let n = 1; n <= 500; n++) {
  // Simulate Coin Flip 50/50 chance
  if (Math.random() < 0.5) {
    numHeads++;
  } else {
    numTails++;
  }
}

// Output Final Results
console.log(`Heads: ${numHeads} Tails: ${numTails}`);

Repeat Unknown # of Times

While loops often work best when repeating code an unknown number of times.

In the example below, a while loop is used to simulate flipping a coin until 100 heads have been flipped.


// Coin Flip Simulator
let numHeads = 0;
let numTails = 0
let numFlips = 0;

// Loop until 100 heads
while (numHeads < 100) {
  // Simulate Coin Flip 50/50 chance
  if (Math.random() < 0.5) {
    numHeads++;
  } else {
    numTails++;
  }
  // Increment numFlips
  numFlips++;
}

// Output Final Results
console.log(`Number of Coin Flips: ${numFlips}`);
console.log(`Heads: ${numHeads} Tails: ${numTails}`);

Summary

  • For loops are commonly used to repeat code a set number of times.
  • While loops are commonly used to repeat code an unknown number of times.
