Local & Global Scope

Variable scope refers to the visibility of a variable: where the variable may be accessed in the program.

A variable declared inside of a function has local scope (only visible inside that function).

A variable declared outside of any functions has global scope (visible everywhere).


let name = 'Megan'; // Global Variable

function displayGreeting() {
  let timeOfDay = 'Morning'; // Local Variable

  console.log('Hello ' + name + '.'); 
  // OK, global variable visible everywhere
  
  console.log('Have a good ' + timeOfDay + "!"); 
  // OK, local variable visible in function
}

displayGreeting();
console.log('Goodbye ' + name + '.'); 
// OK, global variable visible everywhere

console.log('See you next ' + timeOfDay + '.'); 
// ERROR, timeOfDay is local to function
