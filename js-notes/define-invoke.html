Function = Block of Code

A function may be thought of as a named block of code. It stores code for future use.

You define a function as a block of code. When you want to execute the code in the function you must invoke/call the function.

Data may be passed into and out of a function.

Functions help to make code more organized and efficient.

Define & Invoke Functions

See below for an example of how to define and invoke a function.


// Define a Function
function displayGreeting() {
  console.log('Hello.');
  console.log('Have a good day!');
}

// Invoke Function 3 times
displayGreeting();
displayGreeting();
displayGreeting();

To define a function, use the keyword function, then the name of the function followed by parentheses, displayGreeting(). Finally, put the code of the function between opening and closing braces, { }.

To invoke a function, simple use the name of the function followed by parentheses, displayGreeting(). This will tell the function to execute the code stored inside of it.

Caution! ... Common Mistakes

  • When defining a function, remember to have opening and closing braces, { }, to control where the function starts and stops.
  • When invoking a function, don't forget the parentheses, ( ), after the function name.
